# A reference Albedo Participating Node implementation in JavaScript

This is a reference implementation of an Albedo Participating Node (PN) in JavaScript. Albedo
is an automated Matrix federation latency monitoring system. This is a simple implementation of
a standalone PN per the v1.0 specification.

This implementation does not feature observation or data collection features.

## Usage

To run the PN, you will need to have Node.js installed. You can then install the dependencies
and start the PN using the following commands:

```bash
pnpm install
pnpm start
```
