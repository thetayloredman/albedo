# A reference Albedo Central Station implementation in JavaScript

This is a reference implementation of an Albedo Central Station (CS) in JavaScript. Albedo
is an automated Matrix federation latency monitoring system. This is a simple implementation of
a standalone CS per the v1.0 specification.

## Usage

To run the CS, you will need to have Node.js installed. You can then install the dependencies
and start the PN using the following commands:

```bash
pnpm install
pnpm start
```
